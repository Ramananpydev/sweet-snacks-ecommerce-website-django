Sweet & Savory Django Project

A simple and clean Django web application for a Sweet & Savory product shop.
The project includes user authentication, enquiry form with email response, product pages, and MySQL database integration.

Features

Django-based product website

User signup and login with password hashing

Session-based authentication

Contact enquiry form with email reply and PDF attachment

Product and order pages

MySQL database models

Clean URL routing and modular structure

Tech Stack

Python

Django

HTML / CSS

MySQL

SMTP email integration

How to Run

Install dependencies:
pip install -r requirements.txt

Run migrations:
python manage.py migrate

Start the server:
python manage.py runserver

Folder Structure

views.py – handles all application logic

models.py – database models (Login, customer)

forms.py – Django Forms for signup, signin, enquiry

urls.py – application routes
